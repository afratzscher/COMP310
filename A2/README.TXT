Used mimi.cs.mcgill.ca

NOTES:
- The ready queue structure is in the file "ready_queue.h"
- If more than 3 arguments are given (ie. "Set bobbie Montreal Canadians"), only the first 3 arguments are used (ie. Sets bobbie to "Montreal", not bobbie to "Montreal Canadians")
- INSTRUCTIONS FOR MAKEFILE:
	- type "make all" to make executable mykernel
	- type "make run" to execute mykernel
	- type "make test" to execute mykernel with testfile.txt
	- type "make clean" to remove all .o files and exectubale mykernel
